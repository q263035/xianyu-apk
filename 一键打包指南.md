# 闲鱼商品监控器 - 一键打包 APK 指南

## 📋 准备工作

1. **GitHub 账号** - 如果没有，先注册：https://github.com
2. **项目压缩包** - 已发送给你的 `xianyu-apk-v1.6.0.zip`

---

## 🚀 步骤 1：解压项目文件

在电脑上解压 `xianyu-apk-v1.6.0.zip`，得到 `xianyu-apk` 文件夹。

---

## 🚀 步骤 2：创建 GitHub 仓库

### 2.1 打开 GitHub
访问：https://github.com/new

### 2.2 填写仓库信息

| 字段 | 填写内容 |
|------|---------|
| **Repository name** | `xianyu-apk` |
| **Description** | 闲鱼商品监控器 APK |
| **Public** | ✅ 选中（免费使用 Actions） |
| **Initialize with README** | ❌ 不要选中 |

### 2.3 创建仓库
点击 `Create repository` 按钮

---

## 🚀 步骤 3：推送代码到 GitHub

### 3.1 打开终端/命令行

**Windows：**
- 按 `Win + R`
- 输入 `cmd`
- 按回车

**Mac：**
- 按 `Command + Space`
- 输入 `Terminal`
- 按回车

### 3.2 进入项目目录

```bash
cd 桌面/xianyu-apk
```

或者直接用鼠标拖拽文件夹到终端窗口，会自动填入路径。

### 3.3 执行推送命令

**复制以下命令，替换 `YOUR_USERNAME` 为你的 GitHub 用户名：**

```bash
# 初始化 git
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 闲鱼商品监控器 v1.6.0"

# 关联远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/xianyu-apk.git

# 推送到 GitHub（自动触发 APK 打包）
git push -u origin main --tags
```

### 3.4 输入 GitHub 账号密码

推送时会要求输入：
- **Username**: 你的 GitHub 用户名
- **Password**: 输入 **Personal Access Token**（不是登录密码）

**如何获取 Token：**
1. 访问：https://github.com/settings/tokens
2. 点击 `Generate new token (classic)`
3. 勾选 `repo` 和 `workflow` 权限
4. 点击 `Generate token`
5. **复制 Token**（只显示一次，保存好！）
6. 在终端粘贴 Token（粘贴时不会显示字符，正常现象）

---

## 🚀 步骤 4：查看打包进度

### 4.1 进入 Actions 页面

推送成功后，在 GitHub 仓库页面：
1. 点击顶部 `Actions` 标签
2. 看到 `Build APK` 工作流正在运行（绿色或黄色图标）

### 4.2 查看日志

点击正在运行的任务，可以看到实时日志：
- 前 5 分钟：安装依赖
- 5-15 分钟：下载 Android SDK/NDK（约 2-3GB）
- 15-25 分钟：编译 APK

---

## 🚀 步骤 5：下载 APK

### 方式 A：从 Releases 下载（推荐）

打包完成后（约 20-30 分钟）：

1. 进入仓库页面
2. 点击右侧 `Releases`
3. 找到 `v1.6.0` 版本
4. 点击 `xianyuviewer-1.6.0-debug.apk` 下载

### 方式 B：从 Actions 下载

1. 进入 `Actions` 页面
2. 点击已完成的绿色勾选任务
3. 滚动到页面底部
4. 找到 `Artifacts` 区域
5. 点击 `xianyuviewer-debug` 下载

---

## 📱 步骤 6：安装到手机

### 6.1 传输 APK 到手机

- **微信/QQ**: 发送文件到手机
- **数据线**: 直接复制到手机
- **网盘**: 上传后手机下载

### 6.2 安装 APK

1. 手机找到 APK 文件
2. 点击安装
3. 如果提示"未知来源"，允许安装

## ⚠️ 常见问题

### Q1: 推送时提示权限错误

**解决：**
```bash
# 检查远程仓库配置
git remote -v

# 如果错误，删除重新添加
git remote remove origin
git remote add origin https://github.com/YOUR_USERNAME/xianyu-apk.git
git push -u origin main --tags
```

### Q2: Actions 没有自动触发

**检查：**
1. 仓库是否是 **Public**
2. `.github/workflows/build.yml` 是否存在
3. 推送的标签是否正确（`v1.6.0`）

**手动触发：**
1. 进入 `Actions` → `Build APK`
2. 点击 `Run workflow`
3. 选择 `main` 分支
4. 点击 `Run workflow`

### Q3: 打包失败

**查看日志：**
1. 进入失败的 Actions 任务
2. 查看错误信息
3. 常见原因：
   - 网络问题 → 重试即可
   - 依赖缺失 → 检查 build.yml

**重新打包：**
```bash
# 删除旧标签
git tag -d v1.6.0
git push origin :refs/tags/v1.6.0

# 创建新标签并推送
git tag v1.6.0
git push -u origin main --tags
```

### Q4: APK 安装失败

**检查：**
1. 手机安卓版本 >= 7.0
2. 允许"未知来源"安装
3. 重新下载 APK（可能下载不完整）
