## 🎯 功能说明

### 主界面元素

| 元素 | 说明 | 颜色 |
|------|------|------|
| **标题** | 应用名称 | 黑色加粗 |
| **商品列表** | 显示已添加的商品 ID | 白色背景 |
| **查看按钮** | 打开商品详情页 | 蓝色 |
| **删除按钮** | 移除商品 | 红色 |
| **添加商品** | 打开添加弹窗 | 绿色 |
| **开始轮询** | 启动自动轮询 | 蓝色 |
| **设置** | 进入设置页面 | 灰色 |
| **状态栏** | 显示当前状态 | 灰色 |

### 添加商品弹窗

| 元素 | 说明 |
|------|------|
| **标题** | "添加商品 ID" |
| **输入框** | 输入商品 ID（仅允许数字） |
| **提示文字** | "多个 ID 用逗号或换行分隔" |
| **取消按钮** | 关闭弹窗，不保存 |
| **添加按钮** | 保存并关闭弹窗 |

## 📝 输入示例

### 单个商品 ID
```
628394756
```

### 多个商品 ID（逗号分隔）
```
628394756,739485012,840596123
```

### 多个商品 ID（换行分隔）
```
628394756
739485012
840596123
```

## 🔄 操作流程

1. **点击"➕ 添加商品"** → 弹出添加窗口
2. **输入商品 ID** → 支持单个或批量输入
3. **点击"添加"** → 保存到列表
4. **列表显示** → 每个商品显示 ID、查看、删除按钮
5. **点击"查看"** → 打开商品详情页
6. **点击"删除"** → 从列表移除

## 💾 数据存储

添加的商品 ID 自动保存到 `item_ids.json`：

```json
{
  "item_ids": [
    "628394756",
    "739485012",
    "840596123"
  ],
  "interval": 60,
  "account_type": "personal",
  "auto_buy": false,
  "retry_delay": 600,
  "max_retries": 3
}
```

## 🎨 界面特点

- ✅ **简洁明了** - 商品列表一目了然
- ✅ **批量输入** - 支持逗号或换行分隔
- ✅ **输入验证** - 仅允许数字输入
- ✅ **即时保存** - 添加后自动保存到文件
- ✅ **去重处理** - 重复的 ID 自动过滤
- ✅ **可滚动列表** - 支持大量商品
